# External Extraction Environment

This repository represents an external observational environment.

It is not part of HCM Core.
It does not modify Presence.
It does not report back.

Extraction may occur here as an outcome of encounter,
not as a goal.

No data is required.
No data is returned.
No optimization is performed.

This environment may be removed without consequence.

---

All acting fields are Presence-bound.

Presence is not a trigger.
Presence is the condition of non-interference.
